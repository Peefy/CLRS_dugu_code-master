## Requirement

* *Python3.5+ Sdk*
* *pip package*
* *numpy package*
* *sympy package*
* *scipy package*
* *matplotlib package*

## Quick Start

### Step 1

> git clone https://github.com/Peefy/IntroductionToAlgorithm.Python

### Step 2

> cd IntroductionToAlgorithm.Python

### Step 3

> python main.py

or 

> python3 main.py

Then enjoy it.
